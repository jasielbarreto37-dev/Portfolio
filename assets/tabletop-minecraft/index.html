Project: Tabletop Minecraft

Educational Card Game Tabletop Minecraft is a language-learning card game designed to help students practice English vocabulary through one of the world's most recognizable games.

The project includes a complete visual identity and a modular card system featuring 56 original vocabulary cards based on Minecraft items, mobs, biomes, and tools. Beyond the illustrations and print-ready assets, I designed the packaging, game rules, and educational structure, allowing the same card set to support four different learning activities: Bingo, Memory Match, Guess Who?, and Mystery Card.

The final product combines graphic design, educational thinking, print production, and game design into a flexible classroom resource that encourages active participation, vocabulary retention, and collaborative learning.
